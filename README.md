# Software 3D Renderer

A simple software renderer written in Python using Pygame. The project explores the mathematics behind 3D graphics by implementing the rendering pipeline without relying on a game engine.

## Features

- Perspective projection
- Camera transformations
- Back-face culling
- Triangle clipping
- Basic diffuse lighting
- OBJ mesh loading
- First-person camera controls

## Motivation

I built this project to better understand the mathematics and algorithms used in computer graphics and 3D rendering.

## Technologies

- Python
- Pygame
